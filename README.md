# صراع العقول (MindClash) - 2026 🧠

## 📱 وصف التطبيق
تطبيق أسئلة وأجوبة احترافي مع 500 سؤال في 10 مستويات صعوبة، تصميم عصري فاخر، وإعلانات مكافأة.

## ✨ المميزات
- 500 سؤال في 10 مستويات صعوبة (50 سؤال لكل مستوى)
- تصميم Glassmorphism فاخر
- رسوم متحركة نيون احترافية
- دعم اللغتين العربية والإنجليزية
- إعلانات مكافأة (Rewarded Ads) من Yandex
- نظام حياة ونقاط متكامل
- شاشة عدم اتصال احترافية
- متوافق مع الأجهزة اللوحية
- 5 أسئلة لكل مرحلة
- 3 محاولات لكل سؤال
- مؤقت زمني 60 ثانية لكل سؤال

## 🛠 التقنيات المستخدمة
- Kotlin
- Jetpack Compose
- Material You 2026
- Room Database
- Yandex Ads
- Custom Tabs
